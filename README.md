# Agent Skills

A collection of agent skills for .NET development workflows.

## Install

```bash
npx skills add https://github.com/harboe/skills.git
```

You'll be prompted to install globally or into the current project.

## Upgrade

```bash
npx skills update
```

## Testing

- **dotnet-stryker-mutations** — Run Stryker.NET mutation testing to evaluate test quality, identify weak tests, and kill surviving mutants. Skill definition is in `SKILL.md`; detailed reference is in `dotnet-stryker-mutations/REFERENCE.md`.
